# TechGear Pro - Landing Page

A modern, responsive landing page for TechGear Pro - an online electronics and gadgets store.

## 🎯 Features

- **Clean, Modern Design**: Sleek UI with gradient accents and smooth animations
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- **Interactive Elements**: 
  - Smooth scroll navigation
  - Floating cards with parallax effect
  - Product cards with hover animations
  - Add to cart functionality
  - Newsletter subscription
- **No Frameworks**: Pure HTML, CSS, and JavaScript

## 🚀 Tech Stack

- HTML5
- CSS3 (Custom properties, Flexbox, Grid, Animations)
- Vanilla JavaScript (ES6+)

## 📁 Project Structure

```
techgear-pro/
├── index.html      # Main HTML file
├── styles.css      # All styles
├── script.js       # Interactivity and animations
└── README.md       # This file
```

## 🎨 Design Highlights

- **Color Palette**: Purple/indigo primary with amber accents
- **Typography**: System font stack for optimal performance
- **Animations**: CSS keyframes and JS-powered interactions
- **Layout**: CSS Grid and Flexbox for responsive design

## 🔧 Local Development

1. Clone the repository
2. Open `index.html` in a browser
3. No build process or dependencies required!
